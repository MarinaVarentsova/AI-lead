нужно выводить структурированный результат диагностики.

ВАЖНО:
Результат должен формироваться AI-помощником на основании:
1. 4 ответов пользователя;
2. базы знаний knowledge_base_inobr_ai_consultant_v2.md;
3. правил безопасных ответов из базы знаний.

После ответа на 4-й вопрос:

1. Сохранить ответы в ai_diagnostic_answers.
2. Обновить ai_conversations.status = diagnostic_completed.
3. Вызвать OpenAIService.
4. Передать в prompt:
   - текст базы знаний;
   - ответы пользователя:
     experience_area
     experience_years
     education_type
     goal
   - raw-тексты ответов.
5. Получить от AI краткую структурированную консультацию.
6. Показать её внутри карточки "Диагностика завершена".

Формат результата:

Диагностика завершена

Ваш предварительный результат:

1. Опыт
[короткая интерпретация ответа пользователя]

2. Образование
[короткая интерпретация + ограничение, если есть]

3. Цель
[что означает выбранная цель]

4. Рекомендация
[какое направление предварительно подходит]

5. Следующий шаг
[мягкое предложение получить консультацию специалиста]

Кнопка:
"Получить консультацию"

Запрещено:
- показывать просто "Ваши ответы сохранены";
- сразу уводить в контакт без результата;
- обещать заработок;
- обещать трудоустройство;
- обещать судебный результат;
- обещать скидку;
- предлагать ССТЭ / ДПО, если education_type = school_only.

если education_type = school_only:

Рекомендация должна быть:

"Спасибо за ответы. На основании диагностики специалист ИНОБР сможет подобрать подходящее направление обучения. Чтобы получить персональную консультацию, оставьте удобный способ связи."

Технически:
- не менять БД;
- не менять таблицы;
- не менять справочники;
- использовать уже сохраненные ответы из ai_diagnostic_answers;
- сохранить текст результата в ai_messages как assistant message с step = diagnostic_result;
- после показа результата оставить кнопку "Получить консультацию".

После исправления проверить:
1. пользователь отвечает на 4 вопроса;
2. появляется карточка "Диагностика завершена";

4. результат меняется в зависимости от education_type;

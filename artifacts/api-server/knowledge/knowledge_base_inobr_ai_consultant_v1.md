## О проекте

Этот файл будет содержать полную базу знаний AI-консультанта ИНОБР:
- информацию о программах обучения;
- требования к поступлению;
- стоимость и форматы обучения;
- карьерные треки строительного эксперта;
- ответы на часто задаваемые вопросы.

Для обновления базы знаний замените этот файл на актуальный и перезапустите сервер
(или вызовите `knowledgeBaseService.clearCache()` для горячей перезагрузки).
